# External Handoffs

Manual v0 drop location for copied external handoff packets.

Files placed here remain pending evidence until Narada proper records an admission, defer, or reject decision in `.narada/admission/admission-ledger.jsonl`.

Do not place narada-andrey runtime databases, task history, inbox history, checkpoint databases, rosters, operator-surface bindings, PC-locus state, secrets, tokens, credentials, or private operator preferences here.
