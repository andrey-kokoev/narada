## Mission
Create the first real example fixtures for the examples catalog using the canonical schemas.

## Scope

- top-level `examples/`
- first `context-cases/`
- first mailbox-specialized examples

## Seed Set

Add at least these examples:

1. direct support resolution
2. support thread with commitment extraction
3. obligation-centric internal follow-up

## Requirements

- examples should be executable/assertable where possible
- examples may be marked `draft` only if a runner cannot yet fully evaluate them

## Definition Of Done

- [ ] seed example files exist
