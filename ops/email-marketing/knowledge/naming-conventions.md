## Placeholder Content

### Prefix Rules

| Campaign Type | Prefix |
|---------------|--------|
| Promotional | `PROMO-` |
| Newsletter | `NEWS-` |
| Welcome series | `WELCOME-` |
| Re-engagement | `REENG-` |

### Date Format

Use ISO date: `YYYY-MM-DD` or shorthand `YYMMDD`.

Example: `PROMO-2026-04-25-SPRING`

### Segment Abbreviations

| Segment | Code |
|---------|------|
| All customers | `ALL` |
| VIP | `VIP` |
| Active 90 days | `A90` |
| Lapsed 90+ days | `L90` |

### Forbidden Words

- `TEST` (use internal test process instead)
- `DRAFT`
- `TEMP`

### Examples

✅ Valid: `PROMO-2026-04-25-VIP-SpringLaunch`
❌ Invalid: `Spring Launch Promo` (missing prefix, wrong date format)
