### Template Catalog

| Template Name | Purpose | Typical Audience |
|---------------|---------|------------------|
| Welcome Series | Onboard new subscribers | New Subscribers |
| Product Launch | Announce new product | All Customers, VIP |
| Promotional Sale | Discount or special offer | Active 90, Lapsed 90 |
| Re-engagement | Win back lapsed customers | Winback |
| Newsletter | Regular content update | All Customers |

| Pattern | Example |
|---------|---------|
| Question | "Ready for something new?" |

| Curiosity | "We picked this just for you" |
| Update | "April newsletter is here" |

### Content Guidelines

**Promotional Sale:**
- Lead with the offer
- Include clear expiration date
- One primary CTA

**Welcome Series:**
- Thank the subscriber
- Set expectations for email frequency
- Introduce brand value proposition

**Re-engagement:**
- Acknowledge absence gently
- Offer incentive to return
- Easy unsubscribe option
