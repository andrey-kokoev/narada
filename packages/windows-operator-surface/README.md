# @narada2/windows-operator-surface

Descriptor contracts for Windows operator-surface identity projection, runtime binding evidence, OSL labels, panel payloads, and health states.

The package is descriptor-only. It does not capture live HWNDs, mutate PC runtime state, import generated projections, copy narada-andrey identities, or operate on live windows/processes.

## Included First Services

- Fixture-safe binding liveness classification.
- Binding diagnosis assembly from receiving-Site supplied rows.
- Compatibility runtime-binding projection from receiving-Site supplied evidence.

See `docs/source-inventory.md` for admitted source classification and exclusions.
