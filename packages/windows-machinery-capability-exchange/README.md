# @narada2/windows-machinery-capability-exchange

Descriptor-only admission bundle for adopting reusable Windows machinery package slices into Narada proper.

This package records the package dependency graph, receiving-Site adoption checklist, and non-portable exclusions. It does not admit live shell authority, PC-locus mutation, private MCP client mutation, Windows profile mutation, task lifecycle mutation, external publication authority, or import of narada-andrey runtime state.

## Conformance Report

The `narada.windows_machinery_capability_exchange.conformance_report.v0` contract reports whether the adopted descriptor package set is complete and names any missing package slices. It also preserves shared refusal classes for runtime databases, task/inbox history, checkpoints, rosters, operator-surface runtime state, PC-locus state, secrets, credentials, and live process authority.
