# @narada2/ops-kit

Narada operational shaping and preflight layer.

## Contract

`ops-kit` covers:

```text
user intent -> Narada shape -> readiness -> optional operation
```

It is Narada-specific. It should not become a generic installer or config-management system.

## Commands

- `want-mailbox`
- `want-workflow`
- `want-posture`
- `setup`
- `preflight`
- `inspect`
- `explain`
- `activate`
