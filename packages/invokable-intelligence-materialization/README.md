# Invokable intelligence materialization

Portable node:sqlite and Cloudflare D1 embodiments of Narada's governed
cross-locus materialization contract. Origin authority is preserved in every
projection and audit event; destination storage only records an explicit
admission. Resolver acquisition excludes revoked, superseded, expired, and
out-of-scope projections. Request-scoped envelopes are verified through a
runtime-neutral digest/signature interface whose signed binding covers the
request, destination, payload digest, validity interval, and verifier key.

The package stores references and digests only. It never transports payload or
secret values and does not synchronize databases or transfer authority.
