# @narada2/agent-web-ui

Deprecated migration predecessor.

Production launch, attach, and Cloudflare projection paths use
`@narada2/agent-web-ui2`. Do not add new behavior here. Build, test, and run the
replacement from [`../agent-web-ui2`](../agent-web-ui2/README.md).

This package does not publish the `narada-agent-web-ui` binary or a `start`
script. Its remaining source and verification scripts exist only for bounded
migration comparison.
